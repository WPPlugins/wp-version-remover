?ver Remover! Remove wp version param from any enqueued styles & scripts.

== Description ==

It's a very simple plugin for remove wp version param from any enqueued scripts OR styles. Generaly user get issue during update css/js files but it not reflect in front-end due version param. This plugin help you to resolve your this type of all issues.

Note : please don't forget to leave your valuable feedback and suggestions for make this plugin much more better compare to other plugins.

Send your query on raghunath.0087@gmail.com


= `Features` = 
 * An option for remove version param from any enqueued scripts
 * An option for remove version param from any enqueued styles
 * An option for remove generator tags from head section

== Installation ==

Installing the plugin:

1.  Unzip the plugin's directory into `wp-content/plugins`.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  The settings page will be available under settings menu (Settings >> WP Version Remover)
